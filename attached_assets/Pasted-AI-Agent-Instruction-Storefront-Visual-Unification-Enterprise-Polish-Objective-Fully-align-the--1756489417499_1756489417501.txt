AI Agent Instruction – Storefront Visual Unification (Enterprise Polish)
Objective

Fully align the storefront visual design with the ShopLynk landing page for a cohesive, premium, and enterprise-ready user experience. This includes buttons, gradients, backgrounds, typography, shadows, and spacing.

Governance Rules

No functional changes — visuals, styles, and layout polish only.

All tokens must be centralized in themeConfig.js for long-term maintainability.

Match ShopLynk landing page design language for brand consistency.

Accessibility must be preserved — WCAG AA compliant contrast.

Pixel-perfect alignment between landing page and storefront UI.

Design Requirements

Problem: Storefront still has faint reddish undertones compared to the icy-blue ShopLynk landing page.

Solution: Apply the same landing page gradient globally.

Header wrapper

Search/filter bar background

Category filter container
