Align Storefront UI to Landing Page
Objective

Make the storefront header, button styles, global hues, and shadows fully consistent with the ShopLynk landing page for a polished, enterprise-grade visual system.

Governance Rules

No routing or functional changes — visuals only.

Use global design tokens in themeConfig.js for colors, shadows, and radii.

Match landing page branding 1:1 for CTA buttons, highlights, and base hues.

Maintain WCAG contrast compliance.

Implementation Details
1. Global Background & Hue

Apply icy-neutral tones globally to remove reddish undertones:

background: linear-gradient(135deg, #F9FBFF 0%, #F3F7FF 100%);


Apply to:

Header wrapper

Search bar container

Category pill background

2. Back to Dashboard Button

Replicate ShopLynk landing CTA exactly.

background: linear-gradient(135deg, #3B82F6 0%, #60A5FA 100%);
color: #fff;
font-weight: 600;
border-radius: 12px;
padding: 10px 24px;
box-shadow: 0 8px 16px rgba(59, 130, 246, 0.25);
transition: all 0.25s ease-in-out;


Hover State

Acceptance Criteria

✅ Storefront header, buttons, and filters perfectly match ShopLynk landing page styles.

✅ No visual inconsistencies in gradient tones or shadows.

✅ Everything remains fully responsive and accessible.
