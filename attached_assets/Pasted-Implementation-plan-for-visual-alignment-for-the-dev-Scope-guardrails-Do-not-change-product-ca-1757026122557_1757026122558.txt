1) Add shared design tokens (one source of truth)
A. Global CSS variables (recommended)

Create (or extend) a global CSS file that’s loaded app-wide (e.g., src/styles/theme.css), then import it once in your root layout.

:root{
  --brand-from: #3b82f6;  /* blue-500  */
  --brand-to:   #6366f1;  /* indigo-500*/
  --brand-ring: #3b82f6;

  --glass-bg:    rgba(255,255,255,0.70);
  --glass-brd:   rgba(148,163,184,0.35); /* slate-400/35 */
  --glass-blur:  8px;

  --card-brd:    #e2e8f0; /* slate-200 */
  --text-900:    #0f172a; /* slate-900 */
  --text-700:    #334155; /* slate-700 */
  --text-600:    #475569; /* slate-600 */

  --shadow-lg:   0 6px 24px rgba(2,6,23,0.06);
  --shadow-md:   0 4px 16px rgba(2,6,23,0.05);
  --radius:      1rem;     /* base */
  --radius-xl:   1.25rem;  /* ~ rounded-2xl */
}

/* Reusable helpers */
.brand-gradient { background-image: linear-gradient(to right, var(--brand-from), var(--brand-to)); }
.glass {
  background: var(--glass-bg);
  backdrop-filter: saturate(120%) blur(var(--glass-blur));
  border: 1px solid var(--glass-brd);
}
.elev   { box-shadow: var(--shadow-lg); border-radius: var(--radius-xl); }
.ring-brand:focus-visible { outline: none; box-shadow: 0 0 0 4px color-mix(in srgb, var(--brand-ring), transparent 70%); }

B. Tailwind theme (optional but nice)

Extend Tailwind so tokens are available as utilities:

// tailwind.config.ts
export default {
  theme: {

      },

No duplicates: confirm the stray second filter never appears.

No logic changes: filtering, favorites, and product modals behave identically.
