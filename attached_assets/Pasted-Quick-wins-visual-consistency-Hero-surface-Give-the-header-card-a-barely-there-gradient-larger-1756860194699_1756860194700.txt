Quick wins (visual consistency)

Hero surface

Give the header card a barely-there gradient + larger radius for that airy landing-page feel.
rounded-3xl border-slate-200 bg-gradient-to-b from-white to-slate-50 shadow-sm

Container + spacing

Slightly narrower max width for stronger focus: replace max-w-7xl → max-w-6xl (keep on all sections).

Standardize paddings to px-6 lg:px-8 and vertical rhythm to pt-10 pb-8 / py-6.

Chips & pills

Keep all pills the same height and weight: h-8 px-3.5 text-sm font-semibold.

Active pill: bg-blue-600 text-white border-blue-600; Inactive: bg-white text-slate-700 border-slate-200 hover:border-blue-300.

Search bar

Raise to 56px and give a subtle “focus glow”:
h-14 rounded-xl border-slate-200 focus-visible:border-blue-600 focus-visible:ring-0 shadow-[0_1px_2px_rgba(16,24,40,.04)]

Cards (elevation & motion)

Card shell:
rounded-2xl overflow-hidden border border-slate-200 shadow-[0_1px_2px_rgba(16,24,40,.06)] hover:-translate-y-0.5 hover:shadow-[0_8px_24px_rgba(16,24,40,.12)] transition

Image wrapper keeps aspect-square; add a subtle zoom on hover image: group on card + group-hover:scale-[1.02] transition-transform.

Favorite button

Ghost by default with a light hover ring (keeps the UI quiet):
h-9 w-9 bg-white/90 border border-slate-200 hover:ring-1 hover:ring-slate-200

Badges

“Only X left” soften (marketing page uses soft fills):
bg-rose-50 text-rose-700 border border-rose-200
(You can keep “New” as bg-emerald-50 text-emerald-700 border-emerald-200.)

Buttons

Unify primary CTA green: bg-emerald-600 hover:bg-emerald-700

Outline: border-slate-200 hover:bg-slate-50 (everywhere).

Icon sizing

Lock to 14–16px for chips/badges/buttons for that neat, compact feel.

Empty state & loaders

You’re already using them—keep shadows light and type medium; it matches the landing vibe.

Optional UX niceties (fast wins)

Add rel="noopener noreferrer" to all external links (you already do in social).

Persist filter/sort/search in the URL (query params) so refresh/deep link keeps state.

JSON-LD (Product) on the product modal page for SEO (price/brand/availability).

If product count grows: add lazy “Load more” (append) at ~24 cards.
